# 顏色

現在新的字型看起來很讚！

無論如何，當涉及到字型的顏色，

黑色不是你唯一的選項，

在CSS中，`color`屬性設定了CSS選擇器的字型顏色

```
h1 {
  color: red;
}
```

CSS配備了140種顏色，

就像上面使用的`red`。

在很多種情況下，

這些顏色已經足夠了。

無論如何，想要更多種顏色的網頁開發者能使用十六進位和RGB色值。

1. 十六進位色值(#RRGGBB)：      

    十六進位變數代表紅、綠、藍的混和色。
    
    舉例來說，紅色可用十六進制色碼`#FF0000`表示：
        
        `ff`的值代表紅色，
        
        `00`代表綠色，
        
        然後`00`代表藍色。
